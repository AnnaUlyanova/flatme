# goFlat

A goFlat management app

# Features

1. Document management
  - bills
  - contracts

2. Notes
  - housework
  - maintainence

3. Calendar
  - events
  - inspections
  - bills due
  - housework

4. Contacts
  - flatmates
  - landlord
  - emergency contacts

5. Contract generator

6. Leave a reference

7. WebRTC
  - flat meetings

# MVP
- Home
- Join
- User
  - Settings
- Flat (Component inside home?)

- Facebook authentication w/ sessions

## Setup

After cloning this repo:

```sh
npm install
```

To run the app:

```sh
npm run dev
```

To run tests:

```sh
npm run test
```

To run the server only:

```sh
npm start
```
